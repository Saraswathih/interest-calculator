# Interest Calculator Web Application

## Project Overview
This is a full-stack Interest Calculator & Loan Management Web Application.
It helps track:
- Customers & loans
- Investors & returns
- Given / Taken transactions
- Interest calculation
- Portfolio & dashboard summary

## Tech Stack
Frontend:
- React (Vite)
- CSS

Backend:
- Node.js
- Express.js
- MongoDB (Mongoose)

## Folder Structure
- backend → Express + MongoDB server
- frontend → React UI
- seed.js → Demo data seeding script

## Prerequisites
Make sure these are installed:
- Node.js (v18 or above)
- MongoDB (local or MongoDB Compass)
